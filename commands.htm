命令语言

Evolver 有一个用于输入命令的提示符。大多数工作由 交互式图形命令 和 主提示命令 的组合完成。Evolver 提示符为： 

   Enter command:
命令可以是：
  • 单字符图形命令。
  • 主提示命令（通常为全大写）。
  • 用户定义的命令/过程。
在命令提示符处可使用许多通用功能。

命令输入

在 Evolver 运行时，通常可以从键盘输入命令。但在某些情况下（如在读取文件中的命令时），键盘输入不可用。为了在这些情况下也能获取输入，Evolver 使用了 read 命令。

命令分隔符

命令可以用分号 ; 分隔。 这允许多个命令出现在一行上。 最后一个命令不需要分号。 分号后的空格不是必需的，但可以提高可读性。 例如： 
   g 5; r; g 5

复合命令

用户可以在大括号内定义复合命令： 
   define name command_body
   name ::= command_body
其中 command_body 是用分号分隔的一系列命令。 define 形式将命令体作为字面字符串， 而 ::= 形式会立即扩展所有变量。 在 define 中，展开变量要使用 $ 前缀。 例如： 
   define evolve "r;g 5;r;g 5;hessian"
   height ::= 2.3
   evolve
   evolve2 ::= { r; g 5; r; g $height }

命令重复

数字后跟命令将命令重复执行指定次数： 
   5 r
将执行五次 r 命令。

赋值命令

有多种类型的赋值命令可用。它们的一般形式是： 
   element attrib expr
或者 
   element attrib := expr
例如： 
   vertex[3] x 0.5
   vertex[3].x := 0.5

永久赋值

::= 运算符用于"永久"赋值。普通 := 赋值的值在重新加载数据文件时会丢失， 而 ::= 赋值的值会被保留。这对于保持脚本和过程很有用。

用户定义的命令/过程/函数

用户可以在 Evolver 中定义自己的命令。请参阅 过程。

变量赋值

数值变量赋值使用 := 运算符： 
   variable := expr
例如： 
   total := 0
   for ( i := 1 ; i <= 10 ; i++ ) do total += i

局部作用域

使用 local 关键字声明局部变量。局部变量仅在其声明的命令或过程块内可见。 例如： 
   local i,j,x
   for ( i := 1; i <= 10 ; i++ ) do
   { x := i*i;
     printf "%d squared is %d\n",i,x;
   }

重定向和管道输出

输出可以重定向到文件，类似于 Unix shell 重定向。使用 >> 追加到文件， 使用 >> 创建新文件（或覆盖现有文件）。 语法： 
   command >> filename
   command >>> filename
其中 > 表示写入（覆盖），>> 表示追加。 例如： 
   list vertex >> "vertices.txt"
   printf "%f\n",total_energy >>> "energy.log"

控制结构

Evolver 有几种控制结构，语法类似 C 语言。

if-then-else 语句 

   if ( condition ) then command
   if ( condition ) then command else command
   if ( condition ) then { commands } else { commands }
注意 then 和 else 是必需的关键字。 例如： 
   if ( total_area > 10 ) then refine edges where length > 0.1
   if ( pressure > 0 ) then g 5 else hessian

while 循环 

   while ( condition ) do command
   while ( condition ) do { commands }
当条件为真时重复执行命令。 例如： 
   while ( max(vertex,abs(z)) > 0.001 ) do { r; g 10; r }
do-while 循环 
   do { commands } while ( condition )
类似于 while 循环，但至少执行一次命令块。 例如： 
   do { g 10; r } while ( max(vertex,abs(z)) > 0.001 )

for 循环 

   for ( init ; condition ; increment ) do command
   for ( init ; condition ; increment ) do { commands }
标准的 C 风格 for 循环。循环变量必须在循环外声明。 例如： 
   for ( i := 1 ; i <= 10 ; i++ ) do
   { body[i].target := body[i].target + 1;
     printf "Body %d target: %f\n",i,body[i].target;
   }
FOREACH 循环 
   foreach element_type variable do command
   foreach element_type variable where condition do command
遍历指定类型的元素。可选的 where 子句可以过滤元素。 例如： 
   foreach vertex v do v.x := 2*v.x
   foreach facet ff where color == red do dissolve ff
   foreach edge ee where ee.valence == 5 do refine ee

BREAK 

   break
跳出当前的 while、for 或 foreach 循环。

CONTINUE 

   continue
跳过当前迭代的剩余部分，继续循环的下一次迭代。

RETURN 

   return expr
   return
从过程中返回一个值。如果没有表达式，则不返回值。

元素生成器

元素生成器用于指定一组元素（顶点、边、面或体）。 生成器的语法： 
   element_type id
   element_type id_list
   element_type [ array ]
   element_type where condition
   element_type generator attribute
例如： 
   vertex 3
   vertex {1,3,5,7}
   vertex [some_array]
   vertex where x > 0
   edge where color == red
   facet where not fixed
   body where target > 0

ABORT

主提示命令。 中止读取数据文件。 
   ABORT
中止读取数据文件，但保留当前已加载的表面。通常用于调试数据文件中的错误。

ADDLOAD

主提示命令。 从文件加载额外的表面元素。 
   ADDLOAD filename
将指定数据文件中的元素添加到当前表面。不执行新数据文件的读取部分。有关完整说明，请参阅 数据文件读取部分。

AREA

主提示命令。 计算并显示当前面积/能量。 
   AREA
显示当前表面的总面积和能量值。

AREAWEED

主提示命令。 删除面积小于给定值的面。 
   AREAWEED expr
删除所有面积小于 expr 的面及其未使用的边和顶点。

BINARY_PRINTF

主提示命令。 以二进制格式输出。 
   BINARY_PRINTF string,expr,expr,...
使用标准 C fwrite 函数以二进制格式写入数据。格式字符串指定每个参数的数据类型。

BODY_METIS

主提示命令。 使用 METIS 库对体进行分区。 
   BODY_METIS expr
将体分区为 expr 个部分，使用 METIS 图分区库。

BREAKPOINT

主提示命令。 设置调试断点。 
   BREAKPOINT expr
在指定行号设置断点。当执行到达该行时，Evolver 将暂停并显示调试提示符。

CHDIR

主提示命令。 更改当前工作目录。 
   CHDIR stringexpr
将当前工作目录更改为指定的路径。类似于 Unix 的 cd 命令。

CLOSE_SHOW

主提示命令。 关闭图形显示窗口。 
   CLOSE_SHOW
关闭当前打开的图形显示窗口。

CONVERT_TO_QUANTITIES

主提示命令。 将旧式能量和体积约束转换为量形式。 
   CONVERT_TO_QUANTITIES
将数据文件中的旧式能量和体积约束转换为命名量和方法实例。这允许更灵活地使用高级优化技术。

DEFINE

主提示命令。 定义属性、数组、命令等。 
   DEFINE element_type attribute type
   DEFINE array_name type[size]
   DEFINE command_name "command_string"
定义新的元素属性、数组或用户命令。例如： 
   define vertex attribute weight real
   define edge attribute my_attr real[3]
   define my_array real[10][10]
   define evolve "r; g 5; r; g 5"

DELETE

主提示命令。 删除元素。 
   DELETE generator
删除指定的元素及其依赖元素。例如： 
   delete vertex where x < 0
   delete edge where color == red
   delete facet where id > 100

DELETE_TEXT

主提示命令。 删除显示的文本。 
   DELETE_TEXT expr
删除指定编号的显示文本。

DETORUS

主提示命令。 将环面模型转换为普通模型。 
   DETORUS
将环面周期性模型中的表面展开为普通表面。用于可视化或进一步分析。

DIRICHLET

主提示命令。 使用 Dirichlet 能量进行最小化迭代。 
   DIRICHLET
   DIRICHLET expr
使用 Dirichlet 能量（梯度的平方积分）进行一步 Newton 迭代。可选参数限制步长大小。适用于具有固定边界的共形映射。

DIRICHLET_SEEK

主提示命令。 沿 Dirichlet 能量的下降方向搜索。 
   DIRICHLET_SEEK
在 Dirichlet 能量的负梯度方向上进行线搜索以最小化能量。

DISPLAY_TEXT

主提示命令。 在图形窗口中显示文本。 
   DISPLAY_TEXT stringexpr
   DISPLAY_TEXT expr,expr,stringexpr
在图形窗口中显示文本。可以指定位置坐标。

DISSOLVE

主提示命令。 溶解元素。 
   DISSOLVE generator
溶解指定的顶点、边或面。溶解顶点会删除该顶点及其相邻的边和面。溶解边会删除该边及其相邻的面。溶解面会删除该面。例如： 
   dissolve vertex where valence == 2
   dissolve edge where length < 0.001
   dissolve facet where color == red

DUMP

主提示命令。 将当前表面转储到数据文件。 
   DUMP stringexpr
将当前表面状态保存到指定的数据文件中。文件可以稍后加载以继续演化。

DUMP_MEMLIST

主提示命令。 转储内存分配列表。 
   DUMP_MEMLIST
打印所有当前内存分配的列表。用于调试内存问题。

EDGE_MERGE

主提示命令。 合并两条边。 
   EDGE_MERGE expr,expr
合并两条指定的边为一条边。用于简化表面拓扑。

EDGESWAP

主提示命令。 交换边的对角线。 
   EDGESWAP generator
在两个三角形之间交换对角线边。用于改善三角形质量。等同于图形命令 u。

EDGEWEED

主提示命令。 删除短边。 
   EDGEWEED expr
删除长度小于给定值的边及其相邻的面。

EIGENPROBE

主提示命令。 探测 Hessian 的特征值。 
   EIGENPROBE expr
计算 Hessian 矩阵在给定值附近的特征值数量。打印小于、等于和大于给定值的特征值数量。用于检查 Hessian 的正定性。

EPRINT

主提示命令。 打印边信息。 
   EPRINT generator
打印指定边的详细信息，包括端点、长度、相邻面等。

EQUIANGULATE

主提示命令。 对三角形进行等角化。 
   EQUIANGULATE
交换对角线以改善三角形质量。等同于图形命令 u。

ERRPRINTF

主提示命令。 格式化错误输出。 
   ERRPRINTF string,expr,expr,...
类似于 printf，但输出到标准错误流。

EXEC

主提示命令。 执行外部程序。 
   EXEC stringexpr
执行指定的外部程序。类似于 system 命令。

EXPRINT

主提示命令。 打印表达式值。 
   EXPRINT expr
打印表达式的值。类似于 print，但可以用于复杂表达式。

FACET_CROSSCUT

主提示命令。 在面上进行横切。 
   FACET_CROSSCUT expr,expr,expr
在指定面上插入一条横切边，将面分成两部分。

FACET_MERGE

主提示命令。 合并两个面。 
   FACET_MERGE expr,expr
合并两个指定的面为一个面。用于简化表面拓扑。

FIX

主提示命令。 固定元素。 
   FIX generator
固定指定的顶点、边或面。固定元素在演化过程中不会移动。也可以固定参数或量。例如： 
   fix vertex where on_constraint 1
   fix edge where color == red
   fix facet where id < 10
   fix radius
   fix my_quantity

FLUSH_COUNTS

主提示命令。 刷新操作计数器。 
   FLUSH_COUNTS
重置所有操作计数器（如细化、溶解、弹出等）。用于重新开始计数。

FREE_DISCARDS

主提示命令。 释放丢弃的内存。 
   FREE_DISCARDS
释放被丢弃元素占用的内存。通常 Evolver 会自动管理内存，但此命令可用于强制释放。

GEOMPIPE

主提示命令。 通过管道发送几何数据。 
   GEOMPIPE stringexpr
将几何数据通过管道发送到外部程序。用于与其他可视化工具集成。

GEOMVIEW

主提示命令。 启动 Geomview 图形显示。 
   GEOMVIEW
启动 Geomview 几何查看器显示当前表面。需要系统安装 Geomview。

HELP

主提示命令。 显示帮助信息。 
   HELP
   HELP topic
显示 Evolver 的帮助信息。可以指定主题获取特定帮助。

HESSIAN

主提示命令。 执行 Hessian 迭代。 
   HESSIAN
   HESSIAN expr
使用 Hessian 矩阵进行一步 Newton 迭代以最小化能量。可选参数限制步长大小。对于面积最小化问题，Hessian 迭代通常比梯度下降收敛更快。

HESSIAN_MENU

主提示命令。 Hessian 交互式菜单。 
   HESSIAN_MENU
显示 Hessian 操作的交互式菜单。菜单选项包括：
  1. 执行 Hessian 迭代
  2. 显示 Hessian 信息
  3. 计算特征值
  4. 沿特征向量移动
  5. 设置 Hessian 选项
  6. 退出菜单

HESSIAN_SEEK

主提示命令。 沿 Hessian 下降方向搜索。 
   HESSIAN_SEEK
在 Hessian 的负梯度方向上进行线搜索以最小化能量。类似于 saddle 命令。

HISTOGRAM

主提示命令。 显示元素属性的直方图。 
   HISTOGRAM element_type attribute
显示指定元素属性的分布直方图。例如： 
   histogram edge length
   histogram facet area
   histogram vertex x

HISTORY

主提示命令。 显示命令历史。 
   HISTORY
显示最近输入的命令历史。

LAGRANGE

主提示命令。 切换到 Lagrange 模型。 
   LAGRANGE
   LAGRANGE expr
切换到 Lagrange 有限元模型。可选参数指定多项式的阶数。等同于图形命令 M 3 或更高。

LANCZOS

主提示命令。 使用 Lanczos 算法计算特征值。 
   LANCZOS expr
使用 Lanczos 算法计算 Hessian 矩阵的 expr 个最小特征值。用于分析 Hessian 的谱性质。

LINEAR

主提示命令。 切换到线性模型。 
   LINEAR
切换到线性有限元模型。等同于图形命令 M 1。

LIST

主提示命令。 列出元素信息。 
   LIST element_type
   LIST element_type generator
   LIST ATTRIBUTES
   LIST BOTTOMINFO
   LIST PROCEDURES
   LIST TOPINFO
列出指定类型元素的信息。例如： 
   list vertex
   list edge where length > 0.1
   list facet where color == red
   list body
   list attributes
   list procedures

LOAD

主提示命令。 加载新数据文件。 
   LOAD stringexpr
加载新的数据文件，替换当前表面。所有当前状态将丢失。用于开始新的演化。

LOGFILE

主提示命令。 设置日志文件。 
   LOGFILE stringexpr
将所有输入和输出记录到指定文件。用于记录会话。

KEYLOGFILE

主提示命令。 设置键盘日志文件。 
   KEYLOGFILE stringexpr
仅记录键盘输入到指定文件。用于重放会话。

METIS

主提示命令。 使用 METIS 进行分区。 
   METIS expr
使用 METIS 图分区库将表面分区为 expr 个部分。用于并行计算或减少 Hessian 填充。

KMETIS

主提示命令。 使用 K-way METIS 进行分区。 
   KMETIS expr
使用 METIS 的 K-way 分区算法将表面分区为 expr 个部分。通常比普通 METIS 产生更好的分区。

LONGJ

主提示命令。 执行长波长扰动。 
   LONGJ
对表面进行长波长正弦扰动，用于测试表面的稳定性。会提示输入波矢量、相位和振幅。

MATRIX_INVERSE

主提示函数。 计算矩阵的逆。 
   retval := MATRIX_INVERSE(matrix1, matrix2)
计算 matrix1 的逆矩阵并存储到 matrix2。如果矩阵奇异则返回 0，否则返回 1。

MATRIX_MULTIPLY

主提示命令。 计算矩阵乘积。 
   MATRIX_MULTIPLY(matrix1, matrix2, matrix3)
计算 matrix1 和 matrix2 的乘积并存储到 matrix3。

MOVE

主提示命令。 沿当前运动方向移动。 
   MOVE expr
沿之前的运动方向移动表面，步长由 expr 给定。之前的运动方向可以来自梯度步进（g 命令）或 Hessian 步进。

NEW_VERTEX

主提示函数。 创建新顶点。 
   newid := NEW_VERTEX(expr, expr,...)
创建一个新顶点，坐标由参数指定。返回新顶点的 ID。新顶点未连接到任何其他元素。

NEW_EDGE

主提示函数。 创建新边。 
   newid := NEW_EDGE(expr, expr)
创建一条新边，连接两个指定的顶点。返回新边的 ID。

NEW_FACET

主提示函数。 创建新面。 
   newid := NEW_FACET(expr, expr,...)
创建一个新面，边界由指定的边定义。返回新面的 ID。

NEW_BODY

主提示函数。 创建新体。 
   newid := NEW_BODY
创建一个新体。返回新体的 ID。体最初没有面，需要使用 set facet frontbody 和 set facet backbody 命令分配面。

NOTCH

主提示命令。 细分高曲率区域。 
   NOTCH expr
细分所有二面角大于给定值的边。等同于图形命令 n。

OMETIS

主提示命令。 使用 METIS 计算排序。 
   OMETIS expr
使用 METIS 库计算 Hessian 分解的排序。已过时，建议使用 metis_factor 开关。

OOGLFILE

主提示命令。 输出 OOGL 格式文件。 
   OOGLFILE stringexpr
将当前表面以 OOGL 格式写入文件。文件名自动添加 .quad 扩展名。

OPTIMIZE

主提示命令。 设置优化模式。 
   OPTIMIZE expr
将梯度下降迭代设置为优化模式，并设置比例因子的上限。在优化模式下，Evolver 会自动调整比例因子以最小化能量。

PAUSE

主提示命令。 暂停执行。 
   PAUSE
暂停脚本执行直到用户按 Enter 键。用于在脚本中给用户查看输出的机会。

PERMLOAD

主提示命令。 永久加载新数据文件。 
   PERMLOAD stringexpr
加载新数据文件，但继续执行当前命令。用于在脚本中重复演化不同参数的表面。

POP

主提示命令。 弹出元素。 
   POP generator
弹出指定的顶点或边。弹出操作会改变表面拓扑以消除非最小配置。例如： 
   pop vertex where valence > 4
   pop edge where valence > 3

POP_EDGE_TO_TRI

主提示命令。 将边转换为三角形。 
   POP_EDGE_TO_TRI generator
将具有四面体端点的边转换为单个面。用于三维泡沫演化中的拓扑变换。

POP_QUAD_TO_QUAD

主提示命令。 四边形拓扑变换。 
   POP_QUAD_TO_QUAD generator
将由四条三重边界的四边形转换为方向相反的四边形。用于三维泡沫演化。

POP_TRI_TO_EDGE

主提示命令。 将三角形转换为边。 
   POP_TRI_TO_EDGE generator
将具有三个四面体顶点的面转换为单条边。用于三维泡沫演化中的拓扑变换。

POSTSCRIPT

主提示命令。 输出 PostScript 文件。 
   POSTSCRIPT stringexpr
将当前表面以 PostScript 格式写入文件。文件名自动添加 .ps 扩展名。用于生成高质量的打印输出。

PRINT

主提示命令。 打印值。 
   PRINT expr
   PRINT stringexpr
   PRINT commandname
   PRINT arrayslice
   PRINT WARNING_MESSAGES
打印表达式、字符串、命令、数组或警告消息。例如： 
   print total_energy
   print "The area is ", total_area
   print datafilename
   print max(edge,length)

PRINTF

主提示命令。 格式化打印。 
   PRINTF string,expr,expr,...
使用标准 C sprintf 函数格式化打印。所有字符串、整数和浮点格式都有效。例如： 
   printf "Energy: %f, Area: %f\n", total_energy, total_area

QUADRATIC

主提示命令。 切换到二次模型。 
   QUADRATIC
切换到二次有限元模型。等同于图形命令 M 2。

QUIT, BYE, EXIT

主提示命令。 退出 Evolver。 
   quit
   quit expr
不带参数时，提示输入新数据文件。按 Enter 返回当前表面，输入 'q' 退出 Evolver，输入其他内容作为新数据文件名。带参数时，立即退出并使用该值作为退出代码。

RAWESTV

主提示命令。 最原始的顶点平均。 
   RAWESTV
对所有顶点进行平均，不考虑体积守恒或约束。不移动边界上的顶点。

RAWV

主提示命令。 原始顶点平均。 
   RAWV
对所有顶点进行平均，不考虑体积守恒，但只平均具有相同约束类型的顶点。不移动边界上的顶点。

READ

主提示命令。 读取命令文件。 
   READ stringexpr
从文件读取命令，就像在键盘上输入一样。命令可以嵌套。

REBODY

主提示命令。 重新计算体。 
   REBODY
在体被颈部夹断断开后，重新计算连接的体。旧体的面被分成边连接的集合，每个集合定义一个新体。

RECALC

主提示命令。 重新计算并重新显示。 
   RECALC
在更改某些变量或内容后重新计算所有内容。Evolver 尝试在做出重大更改时自动重新计算，但并不总是知道。

REFINE

主提示命令。 细分元素。 
   REFINE generator
细分指定的边或面。通过在每条边的中点插入顶点来细分边，在肥皂膜模型中将相邻的面分成两部分。等同于长边细分命令 l。强烈建议之后进行 等角化。

REPLACE_LOAD

主提示命令。 替换加载数据文件。 
   REPLACE_LOAD stringexpr
用新数据文件替换当前表面，而不进行完全重新初始化。用于在不同条件下重复演化表面。

RESET_COUNTS

主提示命令。 重置计数器。 
   RESET_COUNTS
将各种内部计数器重置为 0。计数器包括：fix_count、unfix_count、where_count、equi_count、edge_delete_count、facet_delete_count 等。

REVERSE_ORIENTATION

主提示命令。 反转方向。 
   REVERSE_ORIENTATION generator
反转指定边或面的内部方向。用于使边或面与约束对齐。

RITZ

主提示命令。 Ritz 特征值计算。 
   RITZ(expr,expr)
计算给定值附近的 Hessian 特征值。第一个参数是移位值，第二个参数是子空间维度（所需特征值数量）。

RENUMBER_ALL

主提示命令。 重新编号所有元素。 
   RENUMBER_ALL
根据存储顺序重新分配所有类型元素的 ID 编号。不要在元素生成器循环内使用此命令！

REORDER_STORAGE

主提示命令。 重新排序存储。 
   REORDER_STORAGE
根据额外属性（如 vertex_order_key、edge_order_key 等）重新排序元素数据结构的存储。用于测试执行速度对存储顺序的依赖性。

SADDLE

主提示命令。 鞍点搜索。 
   SADDLE
   SADDLE expr
沿 Hessian 最低负特征值的特征向量寻找最小能量。如果没有负特征值，则表面不变。可选参数限制步长大小。

SET

主提示命令。 设置属性。 
   SET target attribute expr
设置指定元素的属性。目标必须是单个元素或元素编号。属性包括：

  • EDGE 属性：color、length、no_display、no_dump、noncontent、wrap、bare、fixed
  • FACET 属性：color、opacity、no_display、no_dump、noncontent、frontbody、backbody、positive_var、negative_var、orientation、boundaries、constraints
  • VERTEX 属性：color、no_display、no_dump、no_transform、fixed、constraints
  • BODY 属性：color、no_dump、noncontent、fixed

SHELL

主提示命令。 执行系统命令。 
   SHELL stringexpr
将命令发送到操作系统命令处理器。无参数时启动交互式 shell。

SHOW

主提示命令。 显示元素信息。 
   SHOW target
显示指定元素的详细信息。目标可以是单个元素或元素编号。显示的信息包括坐标、拓扑关系、约束、边界等。

SHOW_EXPR

主提示命令。 显示表达式。 
   SHOW_EXPR expr
显示表达式的值和类型。

SHOW_TRANS

主提示命令。 显示变换。 
   SHOW_TRANS body_id
显示指定 body 的变换信息。

SHOWQ

主提示命令。 显示量化量信息。 
   SHOWQ quantity_name
显示指定量化量的详细信息，包括方法实例、值、梯度等。

SIMPLEX_TO_FE

主提示命令。 转换单纯形到有限元。 
   SIMPLEX_TO_FE
将表面从单纯形表示转换为有限元表示。用于将从复杂数据文件导入的简单三角形表面转换为标准 Evolver 格式。

SOBOLEV

主提示命令。 Sobolev 梯度。 
   SOBOLEV element_type
计算 Sobolev 梯度并存储在速度属性中。比单纯梯度更适合优化问题。

SPRINTF

主提示命令。 格式化字符串。 
   SPRINTF(string,args...)
创建格式化字符串。语法与 C 的 sprintf 完全相同。

SUBCOMMAND

主提示命令。 子命令。 
   SUBCOMMAND stringexpr
将命令字符串发送到 graphics 子进程。如需要返回信息，使用 send。

SYSTEM

主提示命令。 执行系统命令。 
   SYSTEM stringexpr
将命令发送到操作系统命令处理器。无参数时启动交互式 shell。

TRANSFORM_DEPTH

主提示命令。 变换深度。 
   TRANSFORM_DEPTH expr
设置变换链的深度限制。

TRANSFORM_EXPR

主提示命令。 变换表达式。 
   TRANSFORM_EXPR stringexpr
设置变换表达式。用于定义非线性变换。

T1_EDGESWAP

主提示命令。 T1 边交换。 
   T1_EDGESWAP generator
执行 T1 边交换操作。用于泡沫演化中的拓扑变换。

UNFIX

主提示命令。 取消固定。 
   UNFIX element_gen
取消指定元素的固定状态。等同于 SET ... fixed 0。

UNSET

主提示命令。 取消设置。 
   UNSET target attribute
取消指定元素的属性设置。

VERTEX_AVERAGE

主提示命令。 顶点平均。 
   VERTEX_AVERAGE
将每个自由顶点移动到其相邻顶点的平均位置。用于平滑表面。

VERTEX_MERGE

主提示命令。 顶点合并。 
   VERTEX_MERGE vertex_gen
将指定的顶点合并到一个顶点中。用于简化表面拓扑。

WHEREAMI

主提示命令。 当前位置。 
   WHEREAMI
显示当前执行位置，包括文件名和行号。用于调试。

WRAP_VERTEX

主提示命令。 环绕顶点。 
   WRAP_VERTEX vertex_expr wrap_expr
应用环绕因子到顶点。环绕必须是当前活动组之一。

ZOOM

主提示命令。 缩放。 
   ZOOM expr
   ZOOM expr,expr,expr,expr,expr,expr
设置图形子进程的观察中心和视图范围。一个参数时，视图缩小并增加额外空间；六个参数时指定 x、y、z 范围。
